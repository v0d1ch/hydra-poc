minUTxOValue :: ProtocolParameters -> TxOut CtxTx Era -> Value #

Calculate minimum value for a UTxO. Note that cardano-api defines a calculateMinimumUTxO function but it is flawed (see NOTE below) and has an unsatisfactory API because it works across multiple era.

This one is specialized to Babbage and therefore, can be pure.
